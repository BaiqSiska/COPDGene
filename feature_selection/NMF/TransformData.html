flip <- c("pre_FEV6", "pre_FEV1")

Flip pre_FEV6 and pre_FEV1. Show the initial histograms.

hist(pheno$pre_FEV6)


hist(pheno$pre_FEV1)


for (a in flip) {
    pheno[, a] <- -1 * pheno[, a]
    minimum <- min(pheno[, a])
    pheno[, a] <- pheno[, a] - minimum
}

Show the transformed histograms.
